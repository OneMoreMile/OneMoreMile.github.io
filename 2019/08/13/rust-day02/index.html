通用编程概念

  • const常量与不可变变量（默认） const MAX_POINTS: u32 = 100_000;
  • Shadowing 隐藏. 相同变量名称被定义多次
  • 隐藏与mut的区别：let可以改变变量类型，mut不可以
  • 元组与数组 let tup: (i32, f64, u8) = (500, 6.4, 1); 解构；数组是定长的
  • 函数，采用snake case命名法
  • 结构体struct
  • :? :#? ,类似与java的 toString的实现
  • trait 特性，特征
  • 关联函数 impl 中实现的，不包含&self参数的方法
  • match 控制流，有点像java中的 switch case, if let 是match的语法糖
  • 迭代器与闭包：
  • spawn 线程, 1:1 绿色线程;
  • move 机制，let s1 = String::from(“hello”); let s2 = s1;
  • Copy trait 与克隆 clone
  • 引用不获取所有权，内存不会被释放，变量仍然可以用
